# todolist-week5

## How to start

### start React

```bash
npm start
yarn start
```

### start Server

```bash
yarn server
npm run server
```

package.json

```json
// ...
  "scripts": {
    "start": "react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test",
    "eject": "react-scripts eject",
    "server": "json-server --watch db.json --port 3001"
  },
// ...
```
